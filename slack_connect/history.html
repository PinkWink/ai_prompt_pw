MCP Slack 서버 설정 방법 안내

단계별로 설명:

  • 1. Slack App 생성 - api.slack.com/apps에서 새 앱 생성
  • 2. 권한(Scopes) 설정 - Bot Token Scopes 추가: channels:read, channels:history, chat:write, im:read, im:write, im:history, users:read, reactions:read, reactions:write
  • 3. 앱 설치 및 토큰 획득 - Bot User OAuth Token (xoxb-...) 복사
  • 4. Team ID 확인 - Slack 브라우저 URL에서 T0XXXXXXX 형태의 ID 확인
  • 5. Claude Code에 MCP 서버 등록 - 명령어: claude mcp add slack -e SLACK_BOT_TOKEN=xoxb-your-token -e SLACK_TEAM_ID=T0XXXXXXX -- npx -y @modelcontextprotocol/server-slack
  • 6. 확인 - Claude Code 재시작 후 슬랙 도구 사용 가능

Facebook 연동을 위한 사전 준비 작업 안내

사용자가 직접 수행해야 할 단계별 작업을 안내:

  • Step 1. developers.facebook.com 접속 → 개발자 등록
  • Step 2. 새 앱 생성 (Consumer/Business 유형) → App ID, App Secret 메모
  • Step 3. Graph API Explorer에서 토큰 발급
    • 필요 권한: user_posts, user_links, user_photos
    • User Access Token 생성 및 복사
  • Step 4. 단기 토큰(1~2시간)을 장기 토큰(60일)으로 교환 필요 → App ID, App Secret, 단기 토큰을 전달하면 Claude가 처리
  • Step 5. App ID, App Secret, User Access Token을 파일로 저장하여 전달

주의사항: 개발 모드에서는 본인 계정에 대해 앱 검수 없이 사용 가능. 2024년 이후 Facebook이 개인 피드 API를 제한했으므로, 불가 시 Selenium/Playwright 스크래핑으로 대안 전환 가능.
